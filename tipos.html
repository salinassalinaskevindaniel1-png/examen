SOFTWARE DE SISTEMAS OPERATIVOS

El software de sistema (o software de base) es el conjunto de programas preinstalados en una computadora que gestionan el hardware, la memoria y los recursos, actuando como interfaz entre el usuario y los componentes físicos. Es fundamental para el funcionamiento del equipo y la ejecución de aplicaciones.

SOFTWARE DE APLICACION

El software de aplicación son programas diseñados para que el usuario final realice tareas específicas en computadoras o dispositivos móviles, como escribir documentos, navegar por internet, editar imágenes o jugar.

MAS INFORMACION
SOFTWARE DE PROGRAMCION

Un software de programación es un conjunto de herramientas que posibilita a los programadores desarrollar aplicaciones informáticas, utilizando para ello diferentes utilidades y lenguajes de programación. Incluyen: Editores de texto, Compiladores, Intérpretes, Enlazadores, Depuradores, así como Entornos de Desarrollo Integrados (IDE) habitualmente con Interfaz Gráfica de Usuario (GUI). .
